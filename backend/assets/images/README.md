# Logo Construmétrica

Logo oficial en `logo-construmetrica.png` (CONSTRU / MÉTRICA, acento rojo).

Referencia visual: manual de marca en `docs/MANUAL-MARCA.pdf`.

Alternativas por entorno:

- `PROFORMA_LOGO_PATH` — ruta absoluta al PNG
- `PROFORMA_LOGO_BASE64` — contenido PNG en base64 (útil en Railway/Docker sin volumen de assets)
