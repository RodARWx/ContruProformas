# Fuentes Gotham — Construmétrica

Las proformas usan **Gotham Black** (títulos y categorías) y **Gotham Book** (datos).

Coloque aquí los archivos con licencia válida:

| Archivo esperado (cualquiera de estos nombres) | Uso |
|------------------------------------------------|-----|
| `Gotham-Black.otf` / `GothamBlack.otf`         | Títulos, encabezados de tabla, categorías |
| `Gotham-Book.otf` / `GothamBook.otf`           | Datos, notas, contacto |

También puede apuntar rutas personalizadas con variables de entorno:

- `PROFORMA_FONT_GOTHAM_BLACK_PATH`
- `PROFORMA_FONT_GOTHAM_BOOK_PATH`

**Importante:** Gotham es una fuente comercial. No la suba al repositorio sin autorización.

En Docker, las fuentes se instalan automáticamente en el contenedor para LibreOffice.
