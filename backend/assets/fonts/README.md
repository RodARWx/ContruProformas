# Fuentes institucionales — Construmétrica

Fuentes del manual de marca (pág. 14), referencia: `docs/MANUAL-MARCA.pdf`

- `Gotham-Black.otf` — títulos, encabezados, categorías, NOTAS, Contacto
- `Gotham-Book.otf` — datos, rubros, totales, notas

Variables de entorno opcionales:

- `PROFORMA_FONT_GOTHAM_BLACK_PATH`
- `PROFORMA_FONT_GOTHAM_BOOK_PATH`

Si no están presentes, el export usa los OTF de esta carpeta; en su defecto, Arial / Arial Black.
